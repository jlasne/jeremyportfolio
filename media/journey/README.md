# Journey videos

Drop the intro video here and it shows in **The Journey** section on the home page.

- **Video:** `hashtag-0.mp4` → served at `/media/journey/hashtag-0.mp4`
- **Poster (optional):** `hashtag-0.jpg` — shown before the viewer hits play

Keep files under GitHub Pages' **100 MB / file** limit (compress with HandBrake,
or host on YouTube / Cloudflare Stream and ask me to switch the player to an
embed).

Until `hashtag-0.mp4` exists, the section shows a tasteful "video coming soon"
placeholder, so the page never looks broken.
